# Heimatplatz.Features.Notifications.Contracts

## Purpose
Provides shared contracts and models for the Notifications feature in the Uno Platform frontend.

## Contents

### Models
- **NotificationPreferenceDto**: Data transfer object for notification preferences
- **LocationFilterModel**: Model for location filter selection

## Dependencies
None - this is a pure contracts library.

## Usage
Referenced by the main Notifications feature and other features that need to interact with notification settings.
