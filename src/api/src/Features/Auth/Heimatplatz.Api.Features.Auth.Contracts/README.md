# Heimatplatz.Api.Features.Auth.Contracts

Contracts-Projekt fuer das Auth Feature der API.

## Inhalt

- **Mediator/Requests**: Request/Response DTOs fuer Authentifizierung
  - `RegisterRequest` / `RegisterResponse`: Benutzerregistrierung

## Verwendung

Diese Contracts werden vom API Handler verwendet und automatisch via OpenAPI fuer den Uno Client generiert.
