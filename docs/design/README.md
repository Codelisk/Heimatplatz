# Heimatplatz - Immobilien App Oberösterreich

Eine moderne Immobilien-Plattform für Oberösterreich mit Fokus auf Einfachheit und Datenschutz.

## Konzept

**Zielgruppe:** Private Käufer und Verkäufer in Oberösterreich

### Kernfunktionen

- **Immobilienarten:** Häuser, Grundstücke, Zwangsversteigerungen
- **Einfache Filter:** Für private Käufer optimiert
- **Anonyme Nutzung:** Suchen ohne Account möglich
- **Minimale Daten:** Account-Erstellung mit wenig Pflichtfeldern
- **Anbieter-Filter:** Nach Privatpersonen oder Maklern filtern
- **Verkäufer-System:** Für Private und Unternehmen

### Design-Prinzipien

- Schwarz-Weiß Design (minimalistisch)
- Mobile-First Ansatz
- Klare Typografie
- Intuitive Navigation
